
Set up project (Online):

1. Select Developer mode in chrome extensions tab.
2. Load unpacked extension from folder CSE591AW_1_Chrome_Extension_Remote.
3. Go to http://ec2-34-213-252-195.us-west-2.compute.amazonaws.com:8082/ landing page.
4. Click on signup to create a new account or login to existing account.
5. Navigate to Stackoverflow page and test interaction logs mentioned in profile page.
6. Click on View event logs in profile page to see past interaction logs.


Set up project (Local):

1. Download and install NodeJs from https://nodejs.org/en/download/
2. Download and install Git from https://git-scm.com/book/en/v2/Getting-Started-Installing-Git
3. Download and install MongoDB from https://www.mongodb.com/download-center#community
4. Download and install Google Chrome from https://www.google.com/chrome/browser/desktop/index.html
5. Create a new database named 'assignment1' in mongo shell.
6. Clone git project from https://github.com/AravindKv805/CSE591Assignment1.git and go to the folder.
7. Run: npm install
8. Run: npm start.
9. Select Developer mode in chrome extensions tab
10. Load unpacked extension from folder CSE591AW_1_Chrome_Extension_Local
11. Go to localhost:8082 to land on home page.
12. Navigate to Stackoverflow page and test interaction logs mentioned in profile page.
13. Click on View event logs in profile page to see past interaction logs.
